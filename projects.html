Insurance Cost Prediction using Multivariate Linear Regression

Built a machine learning model to predict insurance charges using demographic and health-related features. The project applies multivariate linear regression to analyze how multiple variables influence cost prediction. The dataset includes features such as age, sex, BMI, children, smoking status, charges, and region, which were used to train and evaluate the model’s predictive performance.

Key Features

  • Implemented multivariate linear regression model for prediction
  • Performed data preprocessing and feature selection
  • Trained and evaluated using train/test split
  • Compared performance with and without normalization
  • Analyzed feature impact on insurance cost

Results

  • Evaluated using MSE and R² score
  • Normalization improved model stability and convergence
  • Identified BMI and smoking status as key predictors

Python, NumPy, Pandas, Scikit-learn, Matplotlib

Calendar.io - Full-Stack Scheduling Application

Developed a full-stack calendar application with user authentication, event scheduling, and real-time data management. The application enables users to create, search, and manage events with an interactive and responsive interface.

Key Features

  • User authentication with secure login and session management
  • Create, edit, and delete calendar events
  • Event search and filtering by keyword/date
  • Notification system for scheduled events
  • Dynamic calendar interface with real-time updates

HTML, CSS, JavaScript, Node.js, Express.js, Firebase
